# Basic Keylogger Program

## 📌 Project Description

This project is a simple Keylogger program developed in Java for educational and cybersecurity learning purposes.
The program records the keys pressed on the keyboard and stores them in a text file.

## 🎯 Objective

The objective of this project is to understand how keystroke logging works and to learn about potential security risks related to keylogging.

## ⚙️ Features

* Captures keyboard keystrokes
* Logs the pressed keys
* Saves keystrokes into a text file
* Simple implementation using Java

## 🛠️ Technologies Used

* Java
* File Handling
* Java AWT Event Handling

## ▶️ How to Run the Program

1. Download or clone the repository.

2. Open the project in any Java IDE (VS Code / IntelliJ / Eclipse).

3. Compile the program:

   javac SimpleKeyLogger.java

4. Run the program:

   java SimpleKeyLogger

5. Press keys on the keyboard and the program will log them into a file called **keystrokes.txt**.
